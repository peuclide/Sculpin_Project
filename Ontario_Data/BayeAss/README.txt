BAYEASS running tips

open using terminal to navigate to folder ("BA3*" on desktop)

run file using

./BA3 -v yourfile.txt

file needs to be text file saved as UTF - 8

File should be formatted as follows:

scul101	sunset	loc105	326	330
scul101	sunset	loc128	316	324
scul101	sunset	loc11	218	222
scul101	sunset	loc113	138	138
scul101	sunset	loc213	331	331
scul101	sunset	loc2	233	239
scul101	sunset	loc8	102	102
